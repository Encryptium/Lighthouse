Lighthouse

Last Updated May 30, 2022

    Lighthouse is an open source project committed to providing an great platform for creators to reach out to their audience and update them on progress. To start, go to Lighthouse's Github Page. Fork the repository, and you can start putting together your personal Lighthouse website locally.

A Little About Lighthouse

Lighthouse uses the Django web framework, easily making it capable of much more than a blog platform. For example, it can be extended to include an online store, forum, or newsletter.

Getting Started

To get started after forking the Github repository, make sure you set a secret key in the lighthouse/settings.py file. It can be anything, but make sure that secret key remains secret. Change any of the presets or CSS styles as you wish. The Lighthouse templates doesn't come with much, but what it has will get you up and running very quickly. There is an Articles page like the one you see above in the navbar. There is also an about page, which you are reading right now. All of these are customizable. If you want to publish articles in the Articles tab, you should create a user in the Django administration and give them access to publishing articles.

More to Come in the Future

Currently, Lighthouse is still under the developmental stages. There are few features with limited capabilities. Therefore, some of the features that you want may not be available. It's highly likely these features will come in future updates, but currently, you will have to implement a custom solution before those are available.
